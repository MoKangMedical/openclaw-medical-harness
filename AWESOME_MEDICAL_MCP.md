# Awesome Medical MCP & AI Tools

A curated list of the best open-source medical AI tools and MCP servers, curated by [MoKangMedical](https://github.com/MoKangMedical).

## 🏆 Tier 1: Essential Integrations

### [OpenPharma MCP Servers](https://github.com/openpharma-org/)
**50+ specialized MCP servers for pharmaceutical intelligence**
- Drug data: FDA, EMA, DrugBank, ChEMBL, PubChem
- Genomics: ClinVar, GWAS, gnomAD, Ensembl, GTEx
- Proteomics: UniProt, AlphaFold, PDB, STRING, BindingDB
- Cancer: DepMap, cBioPortal, COSMIC
- Pathways: Reactome, KEGG, Gene Ontology, HPO
- Literature: PubMed, bioRxiv, OpenAlex, CrossRef, CORE
- Clinical: NLM Codes, CDC, Medicare, Medicaid
- ⭐ MIT License | TypeScript | Active development

### [Healthcare MCP Server](https://github.com/Cicatriiz/healthcare-mcp-public)
**Node.js MCP server for healthcare data**
- FDA drug info, PubMed, medRxiv, ClinicalTrials.gov
- ICD-10 codes, DICOM metadata, BMI calculator
- Caching, error handling, Swagger UI
- ⭐ MIT License | Node.js | DXT one-click install
