MCP Tools for Medical AI: PubMed, ChEMBL, OpenTargets Integration

Published: April 16, 2026 · Reading time: 10 minutes

Medical AI agents need real data, not hallucinated references. The MCP (Model Context Protocol) standard enables LLMs to query live biomedical databases during reasoning. This guide covers the 5 key free APIs for medical AI development.

What is MCP?

MCP (Model Context Protocol) is a standard for connecting AI models to external tools and data sources. Instead of the model hallucinating a PubMed citation, the MCP tool queries PubMed in real-time and returns verified results.

1. PubMed — Literature Search

Search 36+ million biomedical articles. Two-step process: (1) search for PMIDs, (2) fetch article details.

GET https://eutils.ncbi.nlm.nih.gov/entrez/eutils/esearch.fcgi?db=pubmed&term=myasthenia+gravis&retmax=5&retmode=json
GET https://eutils.ncbi.nlm.nih.gov/entrez/eutils/esummary.fcgi?db=pubmed&id=12345,67890&retmode=json

Rate limit: 3 requests/second without API key, 10/second with key.

Python Implementation

import httpx

def search_pubmed(query: str, max_results: int = 5):
    resp = httpx.get("https://eutils.ncbi.nlm.nih.gov/entrez/eutils/esearch.fcgi",
        params={"db": "pubmed", "term": query, "retmax": max_results, "retmode": "json"})
    ids = resp.json()["esearchresult"]["idlist"]

    resp2 = httpx.get("https://eutils.ncbi.nlm.nih.gov/entrez/eutils/esummary.fcgi",
        params={"db": "pubmed", "id": ",".join(ids), "retmode": "json"})
    return [resp2.json()["result"][pmid]["title"] for pmid in ids]

2. ChEMBL — Drug Compound Data

Query 2.4 million compounds with bioactivity data against 15,000+ targets.

GET https://www.ebi.ac.uk/chembl/api/data/molecule/search.json?q=aspirin&limit=5
GET https://www.ebi.ac.uk/chembl/api/data/activity.json?target_chembl_id=CHEMBL203&standard_type=IC50&limit=10

3. OpenTargets — Target-Disease Associations

GraphQL API for target-disease association scores (0-1) based on genetics, genomics, drugs, and literature.

# Search for target
query SearchTarget($queryString: String!) {
    search(queryString: $queryString, entityNames: ["target"], page: {index: 0, size: 3}) {
        hits { id name description }
    }
}

# Get target-disease associations
query TargetDisease($ensemblId: String!) {
    target(ensemblId: $ensemblId) {
        associatedDiseases(page: {index: 0, size: 5}) {
            rows { disease { id name } score }
        }
    }
}

4. OMIM — Genetic Disease Database

16,000+ entries for Mendelian disease-gene-phenotype relationships. Requires free API key from omim.org.

GET https://api.omim.org/api/entry?mimNumber=254200&apiKey=YOUR_KEY&format=json

5. OpenFDA — Drug Safety Signals

20+ million adverse event reports. Search by drug name, reaction, or date range.

GET https://api.fda.gov/drug/event.json?search=patient.drug.openfda.brand_name:aspirin&limit=5

FAQ

What is MCP in AI?

MCP (Model Context Protocol) is a standard for connecting AI models to external data sources. In medical AI, MCP enables LLMs to query real databases like PubMed and ChEMBL during reasoning, replacing hallucinated references with verified data.

What free APIs are available for drug discovery?

Key free APIs: (1) ChEMBL for 2.4M compound bioactivity records, (2) OpenTargets for target-disease associations, (3) PubMed E-utilities for biomedical literature, (4) OpenFDA for drug adverse events. None require authentication.

How to search PubMed programmatically?

Use NCBI E-utilities: esearch.fcgi to get PMIDs, esummary.fcgi to fetch metadata. Rate limit: 3 req/sec without key, 10 req/sec with key. Example: curl 'https://eutils.ncbi.nlm.nih.gov/entrez/eutils/esearch.fcgi?db=pubmed&term=query&retmode=json'
