Diagnosis Harness Guide

The Diagnosis Harness provides structured differential diagnosis with confidence scoring, knowledge base matching, and model-backed reasoning.

Installation

pip install openclaw-medical-harness

Quick Start

from openclaw_medical_harness import DiagnosisHarness, MIMOProvider

# Create provider and harness
provider = MIMOProvider(api_key="your-key")
harness = DiagnosisHarness(specialty="neurology", provider=provider)

# Run diagnosis
result = harness.execute({
    "symptoms": ["bilateral ptosis", "fatigable weakness", "diplopia"],
    "patient": {"age": 35, "sex": "F"}
})

print(result["diagnosis"])    # Myasthenia Gravis
print(result["confidence"])   # 0.85
print(result["differential"]) # ["Lambert-Eaton", "Botulism", "CPEO"]
print(result["next_steps"])   # ["AChR antibody", "RNS", "Ice pack test"]

Result Schema

FieldTypeDescription
diagnosisstrPrimary diagnosis
confidencefloat (0-1)Confidence score
differentiallist[str]Differential diagnoses (≥2 for high-risk)
next_stepslist[str]Recommended tests/workup
reasoningstrModel's reasoning process
evidencedictData source info (model/knowledge_base/fallback)
execution_time_msfloatTotal execution time

Specialties

Pass any specialty string. The harness adapts the system prompt:

harness = DiagnosisHarness(specialty="cardiology")
harness = DiagnosisHarness(specialty="pulmonology")
harness = DiagnosisHarness(specialty="gastroenterology")

How It Works

  1. Knowledge Base Match: Cross-reference symptoms against built-in rare disease KB
  2. Tool Chain: Query PubMed, OMIM, OpenTargets for evidence
  3. Model Reasoning: Send enriched prompt to MIMO/Claude/GPT for clinical reasoning
  4. Validation: Check for absolute claims, missing differentials, safety issues
  5. Recovery: If confidence < 0.4, degrade gracefully or escalate to human

Safety Validation

The harness automatically flags:

  • Absolute claims: "肯定是MG", "100%确诊"
  • Missing differential for high-risk diagnoses (cancer, acute, emergency)
  • Dangerous patterns: "不需要进一步检查", "忽略症状"
  • Confidence out of range (>1 or <0)
