Getting Started: Build Your First Medical AI Agent in 5 Minutes

This guide walks you through installing OpenClaw-Medical-Harness, configuring the MIMO model, running your first diagnosis, and deploying a demo server.

Prerequisites

  • Python 3.10 or later
  • MIMO API key (get one at platform.xiaomimimo.com)

Step 1: Install

1/5 Install the package 
pip install openclaw-medical-harness

Step 2: Set API Key

2/5 Configure your MIMO API key 
# Linux/Mac
export MIMO_API_KEY="sk-your-key-here"

# Windows
set MIMO_API_KEY=sk-your-key-here

Step 3: Run Your First Diagnosis

3/5 Create a diagnosis agent 
from openclaw_medical_harness import DiagnosisHarness, MIMOProvider

# Create model provider and harness
provider = MIMOProvider(api_key="sk-your-key")
harness = DiagnosisHarness(specialty="neurology", provider=provider)

# Run diagnosis
result = harness.execute({
    "symptoms": ["bilateral ptosis", "fatigable weakness", "diplopia"],
    "patient": {"age": 35, "sex": "F"}
})

# Print results
print(f"Diagnosis: {result['diagnosis']}")
print(f"Confidence: {result['confidence']:.2f}")
print(f"Differential: {result['differential']}")
print(f"Next steps: {result['next_steps']}")
Output:
Diagnosis: Myasthenia Gravis
Confidence: 0.85
Differential: ['Lambert-Eaton syndrome', 'Botulism', 'CPEO']
Next steps: ['AChR antibody test', 'Repetitive nerve stimulation', 'Ice pack test']

Step 4: Try Drug Discovery

4/5 Run drug discovery pipeline 
from openclaw_medical_harness import DrugDiscoveryHarness

harness = DrugDiscoveryHarness(target_disease="NSCLC", provider=provider)

result = harness.execute({"target": "EGFR", "disease": "NSCLC"})

print(f"Target: {result['target']}")
for c in result['candidates']:
    print(f"  - {c['name']}: {c['activity']}")
print(f"ADMET: {result['admet_profile']}")
Output:
Target: KRAS G12C
- Sotorasib (AMG 510): IC50 < 100 nM (KRAS G12C)
- Adagrasib (MRTX849): IC50 ~10 nM (KRAS G12C)
- Divarasib (GDC-6036): IC50 < 10 nM
ADMET: {'absorption': 'moderate to good oral bioavailability...', 'toxicity': 'GI reactions, liver enzyme elevation...'}

Step 5: Deploy Demo Server

5/5 Start the API server 
pip install "openclaw-medical-harness[server]"
MIMO_API_KEY=sk-your-key python demo_server.py

Open http://localhost:8000/docs for the Swagger UI.

API Endpoints

EndpointMethodDescription
/diagnosePOSTMedical diagnosis from symptoms
/drug-discoveryPOSTDrug target screening & ADMET
/healthPOSTPersonalized health management
/health-checkGETServer health status

Example: Call the API

curl -X POST http://localhost:8000/diagnose \
  -H "Content-Type: application/json" \
  -d '{
    "symptoms": ["bilateral ptosis", "fatigable weakness"],
    "patient": {"age": 35, "sex": "F"},
    "specialty": "neurology"
  }'
