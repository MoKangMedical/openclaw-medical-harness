# OpenPharma MCP Integration Guide How to integrate the [OpenPharma](https://github.com/openpharma-org/) ecosystem (50+ MCP servers) with OpenClaw Medical Harness. ## What is OpenPharma? OpenPharma is a collection of **50+ specialized MCP servers** providing AI agents with access to authoritative pharmaceutical and biomedical data sources. All MIT-licensed, all using public APIs. ## Available Servers ### Drug & Regulatory | Server | Data Source | Use Case | |--------|-----------|----------| | FDA | openFDA API | Drug labels, adverse
