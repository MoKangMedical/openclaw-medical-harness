Health Management Harness Guide

The Health Management Harness generates personalized care plans with specific diet, exercise, medication, and monitoring recommendations.

Quick Start

from openclaw_medical_harness import HealthManagementHarness, MIMOProvider

provider = MIMOProvider(api_key="your-key")
harness = HealthManagementHarness(health_domain="diabetes", provider=provider)

result = harness.execute({
    "conditions": ["type 2 diabetes"],
    "lab_results": {"hba1c": 7.2, "fasting_glucose": 130},
    "patient": {"age": 55}
})

print(result["assessment"])  # Risk level, key findings
print(result["plan"])        # Diet: 1600-1800kcal, 55% carb low GI
print(result["adherence"])   # Tracking metrics
print(result["confidence"])  # 0.70

Input Schema

FieldTypeDescription
conditionslist[str]Existing medical conditions
lab_resultsdictLab values (HbA1c, glucose, lipids, etc.)
wearable_datadictSteps, heart rate, sleep data
health_goalstrTarget outcome (e.g., "HbA1c < 7.0%")
patient.ageintPatient age

Result Schema

FieldDescription
assessmentRisk level, key findings, recommendations
planDiet, exercise, medication, mental health specifics
adherence_metricsWhat to track, check intervals
effectivenessEvaluation timepoints, success criteria
confidenceOverall confidence (0-1)
