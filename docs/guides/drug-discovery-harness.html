Drug Discovery Harness Guide

The Drug Discovery Harness provides a structured 4-stage pipeline from target validation to lead optimization, integrated with real biomedical APIs.

Quick Start

from openclaw_medical_harness import DrugDiscoveryHarness, MIMOProvider

provider = MIMOProvider(api_key="your-key")
harness = DrugDiscoveryHarness(target_disease="NSCLC", provider=provider)

result = harness.execute({"target": "EGFR", "disease": "NSCLC"})

print(result["target"])        # EGFR
print(result["candidates"])    # Real compounds with IC50 values
print(result["admet_profile"]) # Absorption, metabolism, toxicity
print(result["confidence"])    # 0.70

The 4-Stage Pipeline

Target Validation → Virtual Screening → ADMET Prediction → Lead Optimization

Stage 1: Target Validation

Validates target-disease association using OpenTargets GraphQL API. Returns association score (0-1), known drugs, and evidence sources.

Stage 2: Virtual Screening

Screens ChEMBL compound database for bioactivity data against the target. Returns compounds with IC50/EC50 values, mechanism of action, and clinical trial status.

Stage 3: ADMET Prediction

Predicts pharmacokinetic properties:

PropertyDescriptionKey Metrics
AbsorptionOral bioavailabilitylogP, solubility, permeability
DistributionTissue penetrationPlasma protein binding, Vd
MetabolismHepatic clearanceCYP3A4/2D6 substrate/inhibitor
ExcretionEliminationRenal vs hepatic, half-life
ToxicitySafety profilehERG, hepatotoxicity, genotoxicity

Stage 4: Lead Optimization

Suggests specific molecular modifications based on ADMET results. Example: "Improve solubility: introduce polar groups. Reduce CYP inhibition: adjust aromatic substitution."
